# 代码块示例

## 基本代码块
```python
def hello():
    print("Hello, World!")
```

## 带语言标识的代码块
```javascript
function hello() {
    console.log("Hello, World!");
}
```

## 不带语言标识的代码块
```
Some code without language
```

## 包含特殊字符的代码块
```python
def special_chars():
    # 这是一个注释
    print("特殊字符：!@#$%^&*()")
```

## 多个代码块
```python
def first():
    print("First block")
```

```javascript
console.log("Second block");
```
